Criar uma aba de dashboard para o admin, esse dashboard será um serviço de análise de dados sobre toda a biblioteca. 
Para isso, será necessário realizar alterações em todo o backend para conseguir extrair as informações a fim de gerar esse dashboard.

Quero as seguintes informações no dashboard
1. Número de empréstimos por semana
2. Empréstimos solicitados / negados / aprovados
3. Livros mais emprestados
4. Percentual de livros restantes na biblioteca sob a quantidade de livros totais

Verifique quais são as alterações necessárias para implementar o dashboard, vamos fazer o desenvolvimento em sprints para não sobrecarregar um prompt apenas.
O objetivo é conseguir implementar sem alterar a estrutura e funcionalidades existentes.

Sprints abaixo:

---

## SPRINT 1: Preparação de DTOs e Estrutura de Dados
**Objetivo**: Criar DTOs específicos para responses de analytics
- Criar `EmprestimosPorSemanaDTO` (semana, quantidade, ano)
- Criar `EmprestimosStatusDTO` (solicitados, negados, aprovados)
- Criar `LivroMaisEmprestadoDTO` (isbn, titulo, autor, qtdEmprestimos)
- Criar `PercentualLivrosDTO` (livrosTotais, livrosDisponiveis, percentual)
- Revisar estrutura de `DashboardStatsDTO` (container geral)

**Entregáveis**: 4 novos DTOs no pacote com.projeto.biblioteca.dto
